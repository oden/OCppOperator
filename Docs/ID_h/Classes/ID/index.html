Introduction

IDクラス。

Discussion

Objective-Cのid型を引数に取るコピーコンストラクタと、 id型への暗黙の型変換のためのoperatorを定義することで、 Objective-CとC++とのスムーズな型変換を実現・・ できてたらいいなあと思って作ってみた(´・ω・`)
